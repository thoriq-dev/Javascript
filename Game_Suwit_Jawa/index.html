Game Sederhana dengan Javascript

Membuat sebuah game sederhana menggunakan konsep-konsep dasar Javascript, Berikut ini adalah spesifikasi dari programnya:

  1. Sebuah Game sederhana yaitu game Suwit Jawa untuk melawan komputer nantinya.
  2. Menggunakan sebuah Popup Box / Prompt untuk menanyakan inputan dari user, untuk memilih Gajah, Semut atau Orang.
  3. Apabila user memilih salah satu, misalnya user memilih Gajah, lalu user menekan OK.
  4. Kemudian Komputer akan secara random memilih pilihannya juga.
  5. Kemudian User menekan Tombol OK, Lalu akan ada sebuah Popup Box muncul untuk menawarkan apakah akan melakukan suwit ulang kembali.
  6. Apabila user menekan OK untuk mengulang kembali maka user dapat menginputkan kembali. Dan komputer akan melakukan suwit kembali secara random / acak.
